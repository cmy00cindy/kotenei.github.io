区域选择

尝试在该区域用鼠标进行拖拽

调用

                    
KM.areaSelector(options)

参数 options

名称 类型 默认值 描述
$container jquery $("body") 容器，默认整个页面
zIndex int 9999 层叠
autoScroll boolean true 拖动时是否自动滚动窗体
isDraw boolean true 拖动时是否呈现范围
callback object 
                            
                                
{
    onStart:$.noop,
    onMove:$.noop,
    onStop:$.noop                        
}
拖动开始，移动和停止的回调函数
