Ajax

调用

                    

//get提交
kotenei.ajax.get(url,data,[config])
       .done(function(returnStatus){
            ... ajax success    
        })
       .fail(function(){
            ... ajax error      
        })
       .always(function(){
            ... ajax complete       
        });


//post提交，用法跟get相同
kotenei.ajax.post(url,data,[config]);


//表单提交，回调函数跟get相同
kotenei.ajax.ajaxForm($form,[config]);



*ReturnStatus为json格式字符串，须包含以下参数，属性大写开头:
*{
*     Url:String|Null           （跳转页面的地址）
*     Status:Boolean,           （操作成功与否）
*     Message:String|Null,      （操作成功提示信息）
*     ErrorMessage:String/Null, （操作失败错误信息）
*     Data:Object|Null          （返回的数据对象）
*}

参数 config

config参数可不填

名称 类型 默认值 描述
returnUrl object 
                                
{
    enable: true,
    url: location.href
}

enable：是否在ajax地址加上returnUrl参数，默认是

url：returnUrl的地址，默认为当前地址
loadingEnable boolean true 异步操作时是否使用Loading提示，默认使用
popTips object 
                                
{
    enable:true,
    delay: 600,
    immediate: false
}

enable：是否启用弹出提示模块。

delay：弹出提示的显示时间，默认600毫秒。

immediate：是否在弹出提示的回调函数中执行其它操作，如果想在提示隐藏后再执行其它操作，可以设置该值为true，如果不希望等待提示隐藏后再执行其它操作，可以设置为true，默认为false。
