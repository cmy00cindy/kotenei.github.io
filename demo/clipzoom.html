初始化

                    
var clipZoom = new KM.ClipZoom($("div.k-clipZoom"),options);

参数 options

名称 类型 默认值 描述
imgUrl string null 裁剪图片的地址
scale boolean true 是否锁定比例
width int 400 容器宽度
height int 300 容器高度
selectorWidth int 120 选择框的宽度
selectorHeight int 120 选择框的高度
callback object {clip: null} 点击裁剪的回调函数，返回裁剪后的数据
