图片剪裁

HTML结构

                
<div class="k-clipZoom" style="">
    <!--左则选取区-->
    <div class="k-clipZoom-left">
        <div class="k-clipZoom-Box">
            <div class="k-container">
                <img />
            </div>
            <div class="selector">
            </div>
            <span class="zoom" role="plus"><i class="fa fa-plus"></i></span>
            <span class="zoom zoomMinus" role="minus"><i class="fa fa-minus"></i></span>
        </div>
        <div class="button-box">
            <button class="k-btn k-btn-primary" role="clip">裁剪</button>
            <button class="k-btn k-btn-primary" role="center">居中</button>
            <button class="k-btn k-btn-primary" role="reset">重置</button>
        </div>
    </div>
    <!--右则预览区-->
    <div class="k-clipZoom-left" style="margin-left: 10px;">
        <div class="view-box">
            <img />
        </div>
        <div class="result-box">
            <img />
        </div>
    </div>
</div>

初始化

                    
var clipZoom = new kotenei.ClipZoom($("div.k-clipZoom"),options);

参数 options

名称 类型 默认值 描述
imgUrl string null 裁剪图片的地址
scale boolean true 是否锁定比例
width int 400 容器宽度
height int 300 容器高度
selectorWidth int 120 选择框的宽度
selectorHeight int 120 选择框的高度
callback object {clip: null} 点击裁剪的回调函数，返回裁剪后的数据
