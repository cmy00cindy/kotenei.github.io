树型表格

初始化

                    
//创建treeTable对象
var treeTable = new kotenei.TreeTable($elm,options);

参数 options

名称 类型 默认值 描述
expanded boolean true 是否展开所有
className string 'k-treeTable' 树型表格样式名
url string null ajax地址
params object null ajax传递的参数
column int 0 第几列显示树的样式，默认为第1列
columns array [] 列设置 
                                
columns:[
    {
        title: '编号',               //表头名称
        field: 'name',               //字段名
        checkbox: false              //是否checkbox项
        formatter:function(value, data){
            //value:当前节点数据中字段名为name的值，
            //data:当前的节点数据
        }
    },
]
data array [] 数据节点 
                                
data:[
    {
        nodeId: 1,              //节点编号
        parentId: 0,           //父级编号
        [otherParams]               //自定义的一些参数
    },
]
checkType object { "Y": "ps", "N": "ps" }

chkBoxType：关联操作。Y：表示选中，N：表示反选，p:关联父节点,s:关联子节点当前参数只能作用在'checkbox'
