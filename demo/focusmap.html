初始化

                    
KM.focusMap($('.k-focusmap'), options);

参数 options

名称 类型 默认值 描述
containerWidth string null 容器宽度，默认使用图片宽度，可设置'100%'
width int 600 图片宽度
height int 300 图片高度
delay int 3000 图片自动切换速度
