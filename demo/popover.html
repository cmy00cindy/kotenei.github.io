弹出框

左 上 右 下

全局调用

                    
kotenei.Popover.Global();

HTML属性设置

名称 类型 设置值 描述
data-module string popover 告诉程序当前html元素需要初始化为popover模块
data-title string 弹出框标题
data-content string 弹出框内容
data-placement string left,top,right,bottom 弹出框显示的位置
data-trigger string hover,click 触发tips显示和隐藏的事件

示例代码

                
//HTML属性设置
<a  href="javascript:void(0);"  
    data-module="popover" 
    data-title="标题" 
    data-content="上则显示" 
    data-placement="top"
    data-trigger="click">上</a>

自定义

自定义

初始化

                    
var popover = new kotenei.Popover($elm, options);

参数 options

名称 类型 默认值 描述
title string '' 标题
content string '' 内容
placement string 'left' tips显示的方向，可选 'left' , 'top' , 'right' , 'bottom'
trigger string 'hover click' 触发tips显示和隐藏的事件,'manual'则为自定义，通用调用自带函数控制显示和隐藏

函数调用

                    

//设置标题
popover.setTitle('');

//设置内容
popover.setContent('');

//显示
popover.show();

//隐藏
popover.hide();
