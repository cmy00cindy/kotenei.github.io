弹出提示

初始化

                    

kotenei.popTips.success("成功！", function () {
    alert("回调函数");
});

kotenei.popTips.error("失败！", 1500);

kotenei.popTips.warning("警告！", 1500, function () {
    alert("回调函数");
});

参数

名称 类型 默认值 描述
参数1 string 提示内容
参数2 int 3000 tips显示的时间，默认3000毫秒，选填
参数3 function 隐藏后触发的回调函数
