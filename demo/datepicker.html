日期选择

全局调用

                
kotenei.DatePicker.Global();

HTML属性设置

名称 类型 设置值 描述
data-module string datepicker 告诉程序当前html元素需要初始化为datepicker模块
data-format string 'yyyy-MM-dd hh:mm:ss'
'yyyy-MM-dd'等

日期格式化

yyyy：年

MM：月

dd：日

hh：小时

mm：分钟

ss：秒
data-showTime string true or false 是否显示时间选择
data-minDate string '2014-10-01'日期格式或者'today' 最小日期
data-maxDate string '2014-10-01'日期格式或者'today' 最大日期
data-position string 'left' 或者 'right' 日期选择器定位
data-zIndex string '1000'等数值 层叠顺序

示例代码

                
//HTML属性设置
<input  type="text" 
        readonly="readonly" 
        placeholder="全局" 
        data-module="datepicker" 
        data-format="yyyy-MM-dd hh:mm:ss" 
        data-showTime="true" 
        data-minDate="2013-01-11" 
        data-maxDate="today" 
        data-position="right"
        data-zIndex="1000" />

自定义

初始化

                    
//创建datepicker对象
var datePicker = new kotenei.DatePicker($elm,options);

参数 options

名称 类型 默认值 描述
position string 'left' 日期选择器显示的位置，默认与input左对齐，值可以是'left'和'right'
appendTo jQuery $(document.body) 日期选择器生成后插入到的元素，默认插入到body下
showTime boolean false 是否显示时间选择，默认为不显示
year object 
                            
{ 
    min: new Date().getFullYear() - 100, 
    max: new Date().getFullYear() + 100 
}
年份的选择范围
format string 'yyyy-MM-dd'

日期格式化

yyyy：年

MM：月

dd：日

hh：小时

mm：分钟

ss：秒
minDate string null 最小日期，格式'2014-10-11'或'today'
maxDate string null 最大日期，格式同上
zIndex int 2000 堆叠顺序
