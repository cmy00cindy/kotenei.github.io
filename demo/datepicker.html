    日期选择

    全局调用

                
//$elms为带有data-module=datepicker属性的元素，可不填（不填性能会低下）
KM.DatePicker.Global($elms);

    HTML属性设置

    名称 设置值 描述
    data-module datepicker 告诉程序当前html元素需要初始化为datepicker模块
    data-options "{ format:xxx,showTime:xxx ... }" 初始化参数，参照下表 options 说明
    data-onselected 函数名称 选择日期的回调函数
    data-onclean 函数名称 清空日期的回调函数

    示例代码

                
//HTML属性设置
<input  type="text" 
        readonly="readonly" 
        placeholder="全局" 
        data-module="datepicker" 
        data-options="{
            format:'yyyy-MM-dd hh:mm:ss',
            showTime:true,
            minDate:'2013-01-11',
            maxDate:'today',
            position:'right',
            zIndex:1200
        }" />

    自定义

    初始化

                    
//创建datepicker对象
var datePicker = new KM.DatePicker($elm,options);

    参数 options

    名称 类型 默认值 描述
    position string 'left' 日期选择器显示的位置，默认与input左对齐，值可以是'left'和'right'
    appendTo jQuery $(document.body) 日期选择器生成后插入到的元素，默认插入到body下
    showTime boolean false 是否显示时间选择，默认为不显示
    year object 
                                
{ 
    min: new Date().getFullYear() - 100, 
    max: new Date().getFullYear() + 100 
}
    		年份的选择范围
    format string 'yyyy-MM-dd'

    日期格式化

    yyyy：年

    MM：月

    dd：日

    hh：小时

    mm：分钟

    ss：秒
    minDate string null 最小日期，格式'2014-10-11'或'today'
    maxDate string null 最大日期，格式同上
    zIndex int 2000 堆叠顺序

    事件绑定

                
//绑定日期选中事件
datePicker.on('selected',function(date){
    console.log(date);
});
