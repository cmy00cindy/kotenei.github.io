全局调用

                    
//$elms为带有data-module=layout属性的元素，可不填（不填性能会低下）
KM.Layout.Global($elms);

HTML属性设置

名称 设置值 描述
data-module layout 告诉程序当前html元素需要初始化为layout模块
data-options "{ resizeMin : 5 ... }" 初始化参数，参照下表 options 说明
data-onshow 函数名称 面板显示的回调函数
data-onhide 函数名称 面板隐藏的回调函数

示例代码

注：显示/隐藏面板需要加上 role="hide" 属性，data-type 属性是区分面板的方向，必须加上。

                

<!--layout结构开始-->
<div class="k-layout"
        data-module="layout"
        data-options="{ resizeMin:5 }"
        data-onshow="show"
        data-onhide="hide" >

    <!--panel结构开始-->
    <div class="k-panel k-panel-top" data-type="top">
        <div class="k-panel-head">
            <h3 class="k-panel-title ">顶部    
                <span class="fa fa-angle-double-up" role="hide" data-type="top"></span>
            </h3>
        </div>
        <div class="k-panel-body">
            顶部内容
        </div>
    </div>
    <!--panel结构结束-->

    ...

</div>
<!--layout结构结束-->

初始化

                    
var layout = new KM.Layout($elm, options);

参数 options

名称 类型 默认值 描述
resizeMin int 5 调整面板大小时的最小值
cache boolean false 是否缓存当前面板状态 （暂未实现）

事件添加

                
//面板显示时的事件函数
layout.on('show',function(info){
    info: 当前面板的相关信息，包含属性有 width , height , left , top , $panel , $expand 等，详情自使用 console.log 观察
});

//面板隐藏时的事件函数
layout.on('hide',function(info){
    info:参数如上所述              
});
