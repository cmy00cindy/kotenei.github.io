放大镜

HTML结构

                
<div class="k-magnifier">
    <div class="k-magnifier-imgbox">
        <img src="http://d.lanrentuku.com/down/js/tupian-1170/images/small.jpg" 
             data-big-img="http://d.lanrentuku.com/down/js/tupian-1170/images/big.jpg" />
    </div>
</div>

初始化

                    
KM.magnifier($('.k-magnifier'),options);

参数 options

名称 类型 默认值 描述
offset int 10 预览区离图片区的间距
width int 400 图片区宽度
height int 250 图片区高度
zoomWidth int 400 放大宽度
zoomHeight int 400 放大高度
zIndex int 100 层叠顺序
