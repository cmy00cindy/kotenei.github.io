右键菜单

点击右键弹出菜单

初始化

                    
kotenei.contextMenu($elm,options)

参数 options

名称 类型 默认值 描述
className string 'k-contextMenu' 菜单样式名
items array []

菜单项，如：items:[{text:'下载',func:function($target){}}...]
