树型菜单

初始化

                    
//创建tree对象
var tree = new kotenei.Tree($elm,options);

参数 options

名称 类型 默认值 描述
data array [] 节点数据 
                                
data: [
    {
        nodeId: 1,                          //节点编号
        parentId: 0,                        //父节点编号
        rootId: 0,                          //顶级节点编号
        text: '父节点',                     //节点名称
        value: 123,                         //节点值
        isParent: true,                     //是否父级
        open: true,                         //是否打开子级
        nodes:[                             //子级数据
            {
                nodeId: 2, 
                parentId: 1, 
                rootId: 1,
                text: '父节点', 
                value: 456, 
                open: true
            }
        ]
    }
}
check object 
                                
check: {
    enable: false,
    chkType: 'checkbox',
    chkBoxType: { Y: "ps", N: "ps" }        
},

单/复选设置

enable：是否启用

chkType：'checkbox'或者'radio'

chkBoxType：关联操作。Y：表示选中，N：表示反选，p:关联父节点,s:关联子节点当前参数只能作用在'checkbox'
callback object 
                                
callback: {
    beforeCheck: $.noop,
    beforeSelect:$.noop,
    onCheck: $.noop,
    onSelect:$.noop        
},

回调函数

beforeCheck：在勾选单/复选框的前的回调函数

beforeSelect：节点选择前的回调函数

onCheck：勾选单/复选框的回调函数

onSelect：选择节点的回调函数

调用方法

                    
//返回选中的节点
tree.getSelectedNode();

//返回勾选的节点
win.getCheckedNodes();

//添加节点（parentNode：父级节点，nodes：新的节点，节点格式参考data参数格式）
tree.addNodes(parentNode,nodes);

//移除某个节点
tree.removeNode(node);
