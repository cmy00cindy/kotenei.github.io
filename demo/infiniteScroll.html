无限滚动

初始化

                    
var infiniteScroll = new kotenei.InfiniteScroll(options)

参数 options

名称 类型 默认值 描述
$scrollElement jQuery $(window) 滚动容器，默认为浏览器窗口
$watchElement jQuery null 监控元素，监控该元素的底部是否到达滚容器口底部
callback function $.noop 滚动到底部时触发的回调函数
