开关

全局调用

                    
//$elms为带有data-module=switch属性的元素，可不填（不填性能会低下）
kotenei.Switch.Global($elms);

HTML属性设置

名称 设置值 描述
data-module switch 告诉程序当前html元素需要初始化为switch模块
data-values 
                                
"{
    on:{text:'是',value:'1'},
    off:{text:'否',value:'0'}
}"
json字符串格式
data-onClick 函数的名称 点击开关时触发的回调函数，这个属性填函数名就可以

示例代码

                
//html属性设置
<input  type="checkbox" 
        checked="checked" 
        data-module="switch" 
        data-values="{on:{text:'是',value:'1'},off:{text:'否',value:'0'}}"
        data-onClick="test" />

//声明回调函数
function test(value) {
    alert(value);       
}

自定义

初始化

                    
var chkSwitch=new kotenei.Switch($elm,options);

参数 options

名称 类型 默认值 描述
values object 
                                
{
    on: { text: '是', value: true },
    off: { text: '否', value: false }
}
Switch的text和value
callback object 
                                
{
    onclick: $.noop
}
点击Switch时触发的回调函数
