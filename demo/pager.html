分页

初始化

                    
var pager=new KM.Pager($elm,options);

参数 options

名称 类型 默认值 描述
curPage int 1 当前页
totalCount int 0 总记录数
pageSize int 20 分页大小
className string 'k-pagination' 分页样式

事件与函数

                    
//重新生成页码
//如果手动设置了pager.curPage,pager.pageSize,pager.totalCount，可以使用该方法重新生成页码
pager.build();


//点击页码时触发
pager.on('click',function(){
    //函数内可以通过 "this."的方式访问curPage , pageSize , totalCount的值       
});
