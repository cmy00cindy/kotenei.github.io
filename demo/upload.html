全局调用

                
//elms为带有data-module=upload的input和button元素，可不填（不填性能会低下）
KM.Upload.Global($elms);

HTML参数设置

名称 设置值 描述
data-module upload 告诉程序当前html元素需要初始化为upload模块
data-options "{uploadUrl:xxx,removeUrl:xxx ... }" 初始化参数，参照下表 options 说明

示例代码 

                
//html参数设置
<button type="button"
        data-module="upload"
        data-options="{
            uploadUrl:'/',
            removeUrl:'/',
            text:'上传文件'
        }">上传</button>

自定义

初始化

                
var upload = new KM.Upload($elm,options);

参数 options

名称 类型 默认值 描述
uploadUrl string null 上传Url
removeUrl string null 删除Url
$target jquery null 上传或者删除文件后，给目标元素设置val值
text string '上传' 上传按钮文本
name string 'file' 上传控件的name属性值，默认为'file'
uploadedUrls array [] 已上传的文件路径，只能在多文件上传模式初始化时显示
popTips object 
                    
{
    enable: true,
    delay: 600
}

popTips设置

enable：是否启用弹出提示

delay：弹出提示的显示的时间，默认600毫秒
