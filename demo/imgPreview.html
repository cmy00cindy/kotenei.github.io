图片预览

调用

                    
kotenei.imgPreview($elms,options);

参数 options

名称 类型 默认值 描述
delay int 500 渐变显示的速度
backdrop boolean true 是否显示遮罩层
backdropClose boolean true 点击遮罩层是否关闭窗口预览
