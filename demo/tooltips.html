全局调用

                    
//$elms为带有data-module=tooltips属性的元素，可不填（不填性能会低下）
KM.Tooltips.Global($elms);

HTML属性设置

名称 设置值 描述
data-module tooltips 告诉程序当前html元素需要初始化为tooltips模块
data-options "{tipClass:'success',content:'内容' ... }" 初始化参数，参照下表 options 说明

示例代码

                
<a  href="javascript:void(0);" 
    style="font-size: 18px; margin-right: 10px;" 
    data-module="tooltips" 
    data-options="{
        content:'下则显示',
        placement:'bottom',
        tipClass:'danger'
    }">下</a>

自定义

自定义

初始化

                    
var tooltips = new KM.Tooltips($elm, options);

参数 options

名称 类型 默认值 描述
content string '' 提示内容
tipClass string '' 样式名,可选择 'success' , 'warning' , 'info' , 'danger'，默认为黑色
placement string 'left' tips显示的方向，可选 'left' , 'top' , 'right' , 'bottom'
trigger string 'hover click' 触发tips显示和隐藏的事件,'manual'则为自定义，通用调用自带函数控制显示和隐藏

函数调用

                    
//设置内容
tooltips.setContent('');

//显示
tooltips.show('');

//隐藏
tooltips.hide();
