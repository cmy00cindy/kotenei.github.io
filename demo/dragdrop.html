拖放

初始化

                
var dragdrop=new KM.Dragdrop(options);

参数 options

名称 类型 默认值 描述
$layer jQuery object null 被拖动的层
$range jQuery object null 拖动的范围
direction string '' 拖动的方向，'h'为水平,'v'为垂直
boundary boolean false 是否能拖出当前可视窗口
resizable boolean false 拖动层是否能被缩放
scale boolean false 缩放时是否锁定比例
minWidth int 100 缩放时最小宽度
callback object 
                                
{
    start: $.noop,
    move: $.noop,
    stop: $.noop,
    resize:$.noop
}

回调函数

start：开始拖动的回调函数

move：拖动时的回调函数

stop：停止拖动的回调函数

stop：缩放时的回调函数

droppable

试试拖到这里

调用

                
//$elms为带有data-module=droppable属性的元素
KM.Dragdrop.droppable($elms,options);

参数 options

名称 类型 默认值 描述
overClass string 'droppable-over' 滑过的时候添加的样式
callback object 
                                
callback: {
    over: $.noop,
    out: $.noop,
    drop: $.noop
}

over：滑过时的回调函数

out：移出时的回调函数

drop：放置时的回调函数

回调函数返回的参数为 $drag，moveCoord，分别是被拖动的元素和鼠标移动的坐标值，如 
                            
over:function($drag,moveCoord){
    ...
}
