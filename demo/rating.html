评级打分

   

HTML结构

                
<div class="k-rating">
    <div class="k-rating-star"></div>                
</div>

初始化

                    
var rating = new kotenei.Rating($('.k-rating',options);

参数 options

名称 类型 默认值 描述
path string '../images/star' 星星图片所有的路径
starOff string star-off.png 非高亮的星星图片
starHalf string star-half.png 半高亮星星图片
starOn string star-on.png 高亮星星图片
target string .k-rating-target 目标元素选择器
number int 3 星星的个数
values array [1, 2, 3] 每个星对应的值
score int 0 分值,通过分值设置星星高亮
half boolean true 是否显示半星效果
size int 24 星星图片的尺寸
space int 4 星星图片之间的间距

事件回调函数

                    

//score：分值,value：高亮时所对应的value值

rating.on('click', function (score, value) {
    //点击时触发
    $custom.val(score);
}).on('mousemove', function (score, value) {
    //滑过时触发
    $custom.val(score);
}).on('mouseleave', function (score, value) {
    //滑出时触发
    $custom.val(score);
});
