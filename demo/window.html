模态窗体

提示对话框

调用

                    
kotenei.Window.alert('提示框', '警告提示', function () {
    alert('回调');
});

参数

名称 类型 默认值 描述
参数1 string '提示' 对话框标题，可不填
参数2 string 对话框内容
参数3 function 对话框点击确认后的回调函数

确认对话框

调用

                    
kotenei.Window.confirm('确认对话框xxx','您确认要XXXX吗？', function () {
    alert('确认');
}, function () {
    alert('取消');
});

参数

名称 类型 默认值 描述
参数1 string '确认对话框' 对话框的标题，可不填
参数2 string 对话框的内容
参数3 function 对话框点击确认后的回调函数，加上return false;可以阻止窗体关闭。
参数4 function 对话框点击取消后的回调函数，加上return false;可以阻止窗体关闭

全局调用

                
//$elms为带有data-module=window属性的元素，可不填（不填性能会低下）
kotenei.Window.Global($elms);

HTML参数设置

名称 设置值 描述
data-module window 告诉程序当前html元素需要初始化为window模块
data-url 远程地址，iframe为true才会启用
data-id 窗体ID
data-width '1~N'数字 宽度
data-height '1~N'数字 高度
data-iframe 'true' or 'false' 是否启用iframe
data-title 窗口标题
data-content 窗口内容
data-btns 数组变量名称 自定义按钮
data-onok 函数的名称 点击确定的回调函数
data-onclose 函数的名称 关闭窗口的回调函数

示例代码

                
//HTML参数设置
<button class="btn btn-primary btn-lg"
        type="button"
        id="btnGlobal"
        data-module="window"
        data-url="http://www.baidu.com"
        data-width="680"
        data-height="480"
        data-iframe="true"
        data-title="标题"
        data-content="内容"
        data-onok="ok"
        data-onclose="close">
        Launch demo modal
</button>

//设置自定义按钮
var btns: [
    {
        text: '测试',                        //按钮文本
        actionCode: 'test',                 //按钮绑定事件的唯一标识
        className: 'k-btn-success',         //按钮的样式
        func: function () {                 //点击按钮的回调函数
            alert('测试');
            win.close();
        }
    },
]


//设置确定回调函数
function ok() {
    alert("ok");
}

//设置关闭回调函数
function close() {
    alert("close");
}

//全局调用
kotenei.Window.Global($('#btnGlobal'));

自定义

初始化

                    
//创建win对象
var win=new kotenei.Window(options);

参数 options

名称 类型 默认值 描述
url string null 远程地址，如果参数iframe设置为true，则使用iframe模式，否则使用ajax取内容。
params object null 使用ajax取内容时，传递到服务器的参数
id string null 窗体的ID
title string '弹出框' 窗体的标题
content string null 窗体的内容
width int 600 窗体的宽度
height int null 窗体的高度，默认为自适应当前浏览器窗口最大高度，当数值设置过大时，则转为自适应模式，窗体永远在浏览器的正中间。
backdrop boolean true 是否显示遮罩层，默认显示
backdropClose boolean false 点击遮罩层时，是否关闭窗体，默认为不隐藏
iframe boolean false 是否使用iframe模式，设置true的同时，把url也设置上
showFooter boolean true 是否显示窗体的底部区域（按钮区），默认为显示
btns array [] 自定义按钮 
                                
btns: [
    {
        text: '测试',                        //按钮文本
        actionCode: 'test',                 //按钮绑定事件的唯一标识
        className: 'k-btn-success',         //按钮的样式
        func: function () {                 //点击按钮的回调函数
            alert('测试');
            win.close();
        }
    },
]

绑定事件

                    
win.on('ok', function () {
    alert("ok");
}).on('close',function(){
    alert('close');
});

函数调用

                    
//打开窗体
win.open();

//关闭窗体
win.close();

//强制关闭窗体
win.close(true);

//设置标题
win.setTitle('title');

//设置内容
win.setContent('content');
