HTML参数设置

名称 设置值 描述
data-module dropdowntree 告诉程序当前html元素需要初始化为dropdowntree模块
data-data 数据变量的名称 节点数据
data-url 远程数据地址
data-width '1~N' 数字 宽度
data-height '1~N' 数字 高度
data-zIndex '1~N' 数字 层叠顺序
data-isTree 'true' or 'false' 是否树模式，默认为是
data-multiple 'true' or false 是否多选，默认为否
data-bindelement jQuery selector 设置值的关联元素，用jQuery选择器表示，如 '.className' , '#id' 等
data-callback '{select:xxx,check:xxx,hide:xxx}' 回调函数，使用json字符串，xxx为函数的名称

示例代码

                
//节点数据
var nodes = [
    { nodeId: 1, parentId: 0, text: '选项1' },
    { nodeId: 2, parentId: 0, text: '选项2' },
    { nodeId: 3, parentId: 0, text: '选项3' },
    { nodeId: 4, parentId: 0, text: '选项4' },
    { nodeId: 5, parentId: 0, text: '选项5' },
];

//单选回调函数
function select(node) {
    alert('select')
}

//复选回调函数
function check(nodes) {
    alert('check')
}

//HTML参数设置
<input  type="text" 
        class="k-form-control" 
        placeholder="下拉多选列表"
        data-module="dropdowntree"
        data-data="nodes"
        data-istree="false"
        data-multiple="true"
        data-callback="{select:select,check:check}" />

初始化

                
var dropDownTree=new KM.DropDownTree($elm,options);

参数 options

名称 类型 默认值 描述
data array [] 节点数组
url string null 远程数据地址
width int null 宽度，默认与文本框宽度相同
height int 200 高度
zIndex int 999 层叠顺序
isTree boolean true 是否树型模式
multiple boolean false 是否多选
appendTo jQuery object $(document.body) 下拉菜单初始化后插入到哪个元素里，默认是body
callback object 
                            
{
    select:$.noop,
    check:$.noop,
    hide:$.noop                        
}
