全局调用

                    
//$elms为带有data-module=panel属性的元素，可不填（不填性能会低下）
KM.Panel.Global($elms);

HTML属性设置

名称 设置值 描述
data-module panel 告诉程序当前html元素需要初始化为panel模块
data-options "{ width : 'auto',height:'auto' ... }" 初始化参数，参照下表 options 说明
data-onslidedown 函数名称 收起面板的回调函数
data-onslideup 函数名称 展开面板的回调函数

示例代码

                
<div class="k-panel"
    data-module="panel"
    data-options="{height:200}"
    data-onslidedown="slideDown"
    data-onslideup="slideUp">
    <div class="k-panel-head">
        <h3 class="k-panel-title ">这是标题
            <span class="fa fa-angle-double-up" role="slideup"></span>
        </h3>
    </div>
    <div class="k-panel-body">
        这是内容
    </div>
</div>



//展开或者收起的回调函数
function slideDown() {
    alert('slideDown');
}

function slideUp() {
    alert('slideUp');
}

这是内容，可尝试拖动调整面板大小

初始化

                    
var panel = new KM.Layout($elm, options);

参数 options

名称 类型 默认值 描述
width int/string 'auto' 面板宽度
height int/string 'auto' 面板高度
minWidth int 100 最小宽度
minHeight int 100 最小高度
resizable object 
                                
resizable: {
    enabled: false,
    cover: false,
    border: {
        left: true,
        top: true,
        right: true,
        bottom: true
    },
    callback: {
        resize: $.noop,
        stop: $.noop
    }
}

调整大小的设置

enabled：是否启用调整大小，默认为false

cover：调整大小时是否启用覆盖层，默认为启用

border：边框的四条边是否可拖动来调整大小，默认为全部可拖动

callback：调整大小时的回调函数，resize：拖动时触发，stop：停止拖动时触发，函数会返回当前面板相关信息info，如配置 resize:function( info ){ },相关说明请参考 调整大小DEMO

事件添加

                
//面板展开时的事件函数
panel.on('slideDown', function () {
    alert('slideDown');
});

//面板收起时的事件函数

    alert('slideUp');
});
