延迟加载

慢慢滚动观察图片加载


调用

                    
//直接调用（img标签的src设置默认图片，并设置data-url属性存放加载的图片）
kotenei.lazyload();


//自定义参数调用
kotenei.lazyload($elms,options);

参数 options

名称 类型 默认值 描述
placeholder string null 图片占位符路径，当设置该值的时候，图片src会被设置为该值，并把原来的src存放到新增的data-url属性里
$container jQuery $(window) 滚动容器，默认为浏览器窗口
