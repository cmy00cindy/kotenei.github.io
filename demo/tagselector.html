    使用

                        
//elms:不填会从页面上获取带有data-module="tagselector"属性的元素
KM.tagSelector($elms, options);

    HTML属性设置

    名称 设置值 描述
    data-module tagselector 告诉程序当前html元素需要初始化为tagselector模块
    data-onclick 函数名称 点击时回调函数

    参数 options

    名称 类型 默认值 描述
    callback object {onClick:&.noop} 点击时的回调函数 
                                    
onClick:function($el,$layer){
    $el:当前被点击的元素
    $layer：遮罩层元素
}
