字数限制

您还可以输入：140 字符

全局调用

                    
//$elms为带有data-module=wordlimit属性的元素，可不填（不填性能会低下
KM.WordLimit.Global($elms);

HTML属性设置

名称 设置值 描述
data-module wordlimit 告诉程序当前html元素需要初始化为wordlimit模块
data-options { feedback:xxx,maxLength:xxx } 初始化参数，参照下表 options 说明

示例代码

                
//HTML属性设置
<textarea rows="4" 
          cols="50" 
          data-module="wordlimit" 
          data-options="{
            feedback:'#feedback1',
            maxLength:140
          }" ></textarea>

自定义

您还可以输入：20 字符

初始化

                    
var wordlimit = new KM.WordLimit($elm,options);

参数 options

名称 类型 默认值 描述
feedback string '.chars' 剩余字数反馈，值设置jQuery选择器，如'#id'
maxLength int 140 最大字符数
