初始化

                    
KM.scrollImg($('.k-scrollImg'), options);

参数 options

名称 类型 默认值 描述
padding int 30 左右内边距
width int/string 'auto' 容器宽度
height int 150 容器高度
delay int 3000 自动切换的时间间隔
toggleDealy int 700 切换时的速度间隔
